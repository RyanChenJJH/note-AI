# 04 · UI 设计规范 · 晨雾 Sage Mist

> 风格关键词：**安静、舒适、享受**。柔和自然色 + 大量留白 + 细腻微动效（鼠标视差、悬停轻浮、入场淡入）。**不堆科技感、不用杂乱颜色。**
> 设计系统由 `ui-ux-pro-max` 技能推导（Swiss Modernism 2.0 + Wellness 配色）。

## 1. 配色 Token

雾白底、白卡片、雾灰主色、静谧青强调。语义化为 CSS 变量 / Tailwind theme。

| 角色 | Hex | 说明 |
|------|-----|------|
| `--background` | `#F5F5F0` | 雾白页面底 |
| `--card` | `#FFFFFF` | 卡片/面板 |
| `--foreground` | `#0F172A` | 主文字（墨） |
| `--muted-foreground` | `#64748B` | 次文字/说明 |
| `--primary` | `#6B7280` | 雾灰主色（按钮、强调文本） |
| `--primary-foreground` | `#FFFFFF` | 主色上的文字 |
| `--accent` | `#0891B2` | 静谧青（关键 CTA、选中、链接） |
| `--accent-foreground` | `#FFFFFF` | 青色上的文字 |
| `--border` | `#EDEEEF` | 细边框/分隔线 |
| `--muted` | `#F6F6F7` | 浅底块/hover 底 |
| `--destructive` | `#DC2626` | 危险/删除（克制使用） |
| `--ring` | `#6B7280` | 焦点环 |

- 强调色**克制**：青色只用于 CTA、选中态、已导出徽章等关键处，不铺面积。
- 暗色模式：本期可不做；若做，按 `color-dark-mode` 用降饱和的同色系，单独验证对比度。

## 2. 字体

```css
@import url('https://fonts.googleapis.com/css2?family=Lora:wght@400;500;600;700&family=Raleway:wght@300;400;500;600;700&display=swap');
```

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题/封面 | **Lora**（serif） | 有机曲线、温润 |
| 正文/UI | **Raleway**（sans） | 优雅简洁 |
| 中文 | **Noto Serif SC** / **Noto Sans SC** | 与上对应；正文优先 Sans SC |
| 等宽（代码） | JetBrains Mono / ui-monospace | 对话里若含代码块 |

Tailwind：
```js
fontFamily: {
  serif: ['Lora', 'Noto Serif SC', 'serif'],
  sans:  ['Raleway', 'Noto Sans SC', 'sans-serif'],
  mono:  ['JetBrains Mono', 'ui-monospace', 'monospace'],
}
```

- 正文行高 1.6–1.75，行宽 ≤ 75 字符，阅读区充分留白。
- 字号阶：12 / 14 / 16 / 18 / 24 / 32 / 48（封面大标题）。

## 3. 动效（鼠标视差 + 微交互）

核心氛围来自**克制而柔和**的动效。统一时长 **180–280ms**，缓动 ease-out 进、ease-in 出；遵守 `prefers-reduced-motion`（关闭时全部退化为无动画）。

| 场景 | 效果 | 实现要点 |
|------|------|----------|
| 封面青光晕 | 一团青色径向渐变随鼠标**极慢飘移**（视差仅 6–12px） | 监听 `mousemove`，`transform: translate()` + `transition` 缓动；用 `requestAnimationFrame` 节流 |
| 卡片/列表项悬停 | 轻浮 2px + 阴影渐显 + 底色转 `--muted` | `transform: translateY(-2px)`，只动 transform/opacity/shadow |
| 列表入场 | 逐条淡入，stagger 30–40ms | 进入视口触发，避免一次性全弹 |
| 选中态 | 青色细描边 + 勾选打钩淡入 | 不做位移抖动 |
| 切换页/进入 App | 柔和 crossfade（≤300ms） | 维持空间连续性 |
| 已导出徽章 | 静态柔光，不闪烁 | — |

性能纪律：**只动 `transform` / `opacity`**，不动 width/height/top/left；`mousemove` 必须节流。

## 4. 封面页

- 雾白底，居中。背后一团**缓慢呼吸**的雾青色径向渐变球，随鼠标极慢视差。
- 内容：
  - 主标题（Lora 大字）：**AI 对话档案**
  - 副标题（Raleway，muted）：把与 AI 的对话，安静地沉淀成笔记

## 5. 主界面（两栏）

```
┌───────── 顶部细条：标题 · 时间范围(7天/1月/全部) · 导出目录 ─────────┐
├───────────────────────┬───────────────────────────────────────────┤
│ [ Codex ] [ Claude ]   │  会话标题（Lora）   · 来源 · N 轮 · 时间      │
│  ── 点击才加载 ──       │  ─────────────────────────────────────────  │
│                        │  ☐ 第1轮 · 提问                              │
│ ▸ 今天                  │     （人类提问正文，可滚动）                  │
│   · 标题A   ●已导出      │  ☐ 第1轮 · 回复                              │
│   · 标题B               │     （AI 文字回复，可滚动）                  │
│ ▸ 昨天                  │  ☐ 第2轮 · 提问 / 回复 …                     │
│   · 标题C   ◐部分        │                                            │
│ ▸ 5/30                  │  ─────────────────────────────────────────  │
│   · 标题D               │  文件名[____]  ⦿AI整理  [ 导出为 .md ]       │
└───────────────────────┴───────────────────────────────────────────┘
```

- **左栏**：分类 Tab（点击懒加载）→ 时间范围筛选 → 按日期分组的可折叠列表 → 标题带已导出徽章（●已导出 / ◐部分）。
- **右栏**：完整对话阅读区，按轮分块、每块独立 checkbox；底部固定 ExportBar（文件名 + AI 整理开关 + 导出按钮）。
- 已导出的块：标题/块上有柔和徽章，鼠标悬停显示导出时间与目标文件。

## 6. 组件与可达性

- 图标用 **Lucide**（SVG，统一 stroke），**不用 emoji** 当图标。
- 所有可点元素 `cursor: pointer` + 可见 focus 环（青色 2px）。
- 文本对比度 ≥ 4.5:1；颜色不作为唯一信息载体（已导出同时有图标+文字）。
- 点击区 ≥ 32px（桌面），勾选框带足够 hit area。
- 响应式：本工具桌面优先，最小适配 1024px 宽即可。

## 7. 落地提示

- 用 shadcn/ui 组件，主题 token 映射到第 1 节的 CSS 变量。
- 封面光晕、列表 stagger 等动效抽成独立 hook/组件，统一时长与缓动 token。
- 交付前过一遍 `ui-ux-pro-max` 的 Pre-Delivery Checklist（对比度、reduced-motion、focus、hover 过渡）。
