## 3. 接口契约（`docs/06 §3`，原样落地）

```ts
// server/src/ai/tidy.ts
export interface TidyInput {
  title: string;
  source: 'claude' | 'codex';
  blocks: { id: string; role: 'question' | 'answer'; text: string }[];
}
export interface TidyResult {
  markdown: string;           // 整理后正文（不含 front matter）
  suggestedFilename?: string; // AI 自动命名（本期不产出）
  usedAi: boolean;            // 是否真走了模型（占位恒 false=降级）
}
export async function tidy(input: TidyInput, cfg: AiConfig): Promise<TidyResult>;
```

**占位实现**：按块原样拼接，标题用「## 第 N 轮 · 提问/回复」（N 从 `id` 的 `r{N}-` 解析），与非 AI 路 `buildBody` 输出**逐字一致**，保证占位是真正的"无副作用降级"。
```ts
const markdown = input.blocks.map(b => {
  const n = b.id.match(/^r(\d+)-/)?.[1] ?? '?';
  return `## 第 ${n} 轮 · ${b.role === 'question' ? '提问' : '回复'}\n\n${b.text}`;
}).join('\n\n');
return { markdown, usedAi: false };
// TODO(阶段6后迭代)：按 cfg.provider 调真实模型，套 cfg.skill 提示词整理；失败/超时 → 降级返回上面的原样拼接。
```

## 4. 导出链路接入（`routes/export.ts`）

```text
body = { id, blockIds[], filename?, aiTidy? }
cfg = loadConfig(configPath)
{ body: rawBody, involvedRounds } = buildBody(rounds, blockIds)
if (aiTidy):
    result = await tidy(toTidyInput(meta, rounds, blockIds), cfg.ai)
    bodyMd = result.markdown; usedAi = result.usedAi; suggested = result.suggestedFilename
else:
    bodyMd = rawBody; usedAi = false
md = buildFrontMatter(meta, involvedRounds) + "\n\n" + bodyMd + "\n"
filename = body.filename?.trim() || suggested || meta.title.slice(0,20)
out = writeExport(md, dir, filename)
recordExport(..., aiTidy = usedAi)
return { ok:true, path: out, usedAi }
```
- **触发门控**：路由是否走 `tidy()` 由**请求的 `aiTidy`**决定（前端开关即用户每次导出的意图）；`cfg.ai.enabled`/provider 由 `tidy()` 内部判定——占位阶段一律降级。后续真实接入时 `tidy()` 再据 `cfg` 决定真跑还是降级。
- `toTidyInput`：仅取选中且有文本的块，按轮升序、提问在回复前（与 `buildBody` 同序）。

## 5. exporter 重构（输出不变）

- 抽出 `buildBody(rounds, blockIds): { body: string; involvedRounds: number[] }`（即现有 section 循环）。
- 抽出 `buildFrontMatter(meta, involvedRounds, opts?): string`（即现有 front 块；`yamlString`/`localISONow` 私有不变）。
- `buildMarkdown(meta, rounds, blockIds, opts?)` = `${buildFrontMatter(meta, body.involvedRounds, opts)}\n\n${body.body}\n`，**与现状逐字相同** → `exporter.test.ts` 不改照绿。
