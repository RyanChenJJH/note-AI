## 1. 本期目标与范围

**目标**：把空的 `web/` 立起来——技术栈就位、晨雾主题落地、封面页可看可交互、点「进入」柔和切到 `/app` 占位壳。

**范围内**（= `docs/07` 阶段 3）：
- ✅ Vite + React + TS 初始化；Tailwind v4 + shadcn/ui 接入。
- ✅ 晨雾配色 token（`docs/04 §1`）→ CSS 变量 + Tailwind `@theme`；字体（Lora/Raleway + Noto SC）。
- ✅ 封面页：呼吸雾青光晕 + 鼠标极慢视差 + 单一「进入」CTA。
- ✅ 路由：`/`（封面）→ `/app`（两栏壳占位），切换 crossfade ≤300ms。
- ✅ Vite dev 代理 `/api` → `http://127.0.0.1:8787`（落地 `docs/09 §6` 预案）。
- ✅ `lib/api.ts` 薄封装（fetch + 错误归一），先打通 `GET /api/sources` 验证联通。

**范围外**（留给后续阶段）：

- ❌ AI 整理开关真实联动 → **阶段 5**。

- ❌ 暗色模式、列表虚拟化 → 后续迭代。

> `/app` 本期只是**带顶栏 + 两栏骨架占位**的壳（空状态文案），证明路由/主题/布局成立即可，不接数据。

## 3. 技术选型与版本

| 选项 | 取值 | 理由 |
|------|------|------|
| 构建 | **Vite 7 + React 19 + TS** | 阶段 6 要 `vite build` 产物，dev 代理天然支持 |
| 样式 | **Tailwind v4**（`@tailwindcss/vite` 插件，CSS-first `@theme`） | 无 `tailwind.config.js`，token 直接写在 CSS；shadcn 已支持 v4 |
| 组件 | **shadcn/ui**（new-york 风） | `docs/04 §7` 指定；本期只装 `button`，其余阶段 4 按需加 |
| 路由 | **react-router-dom v7** | `docs/02 §5` 的 `/` + `/app`；轻量、标准 |
| 数据 | 本期**裸 fetch**（`lib/api.ts`），React Query 留阶段 4 | 封面壳不需要缓存层，避免过早引入依赖 |
| 图标 | **lucide-react** | `docs/04 §6` 指定，不用 emoji |

依赖（计划安装）：
- 运行时：`react`、`react-dom`、`react-router-dom`、`lucide-react`、`clsx`、`tailwind-merge`、`class-variance-authority`、`tailwindcss-animate`(或 v4 等价)。
- 开发：`vite`、`@vitejs/plugin-react`、`typescript`、`@types/*`、`tailwindcss`、`@tailwindcss/vite`。
- shadcn CLI 通过 `npx shadcn@latest init/add` 落地，不长驻依赖。

## 5. 封面页（`docs/04 §4`）

布局：雾白底、垂直水平居中、单列极简（无导航）。
- `GlowOrb`：背后一团雾青径向渐变球，绝对定位居中、`filter: blur()`、低透明度。
  - **呼吸**：CSS `@keyframes` 缩放 1.0↔1.08 + 透明度微变，周期 ~6s，`ease-in-out`。
  - **视差**：`usePointerParallax` 给 `transform: translate(x,y)`，幅度 6–12px，rAF 节流，`transition` 缓动跟随。
- 文案：主标题「AI 对话档案」(Lora 48)；副标题「把与 AI 的对话，安静地沉淀成笔记」(Raleway, muted)。
- CTA：青色/玻璃感「进入」按钮（shadcn Button + 自定义类）→ 触发 crossfade 后 `navigate('/app')`。
- `prefers-reduced-motion`：呼吸与视差**全部停**（光晕静态居中），CTA 直接跳转无淡出。

## 6. 路由与 crossfade（`docs/02 §5`、`docs/04 §3`）

- `main.tsx` 包 `<BrowserRouter>`；`App.tsx` 定义 `<Routes>`：`/`→`Cover`，`/app`→`AppShell`，`*`→重定向 `/`。
- crossfade（≤300ms）：本期用**轻量 CSS 过渡**实现，不引动画库——
  - 方案：进入/离开页面根节点用 `opacity` 过渡（key 切换触发），或封面点击时先淡出当前再 `navigate`。
  - reduced-motion 时取消过渡，直接切换。
- `/app` 占位：顶部细条（标题占位 + 时间范围/导出目录占位文案）+ 左右两栏空状态（"阶段 4 接入"提示），仅验证布局与主题。

## 7. Vite 代理与 API 封装（`docs/09 §6`）

`vite.config.ts`：
```ts
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import tailwindcss from '@tailwindcss/vite';
import { fileURLToPath, URL } from 'node:url';

export default defineConfig({
  plugins: [react(), tailwindcss()],
  resolve: { alias: { '@': fileURLToPath(new URL('./src', import.meta.url)) } },
  server: { proxy: { '/api': 'http://127.0.0.1:8787' } },
});
```
`lib/api.ts`：
```ts
export async function apiGet<T>(path: string): Promise<T> { /* fetch /api…, 非 2xx 抛错, 解析 json */ }
export const getSources = () => apiGet<{claude:{count:number}; codex:{count:number}}>('/api/sources');
```
- 同源走代理，**无需 CORS**（后端阶段 2 也未开 CORS）。
- 封面壳本期可不实际调用；保留 `getSources()` 供 `/app` 占位做一次"联通自检"（失败则显示离线提示，不报错崩溃）。

## 9. 风险与对策

| 风险 | 对策 |
|------|------|
| Tailwind v4 + shadcn 初始化踩坑（v4 与旧 init 流程不同） | 用 `shadcn@latest`（已适配 v4 的 `@theme`/CSS 变量）；token 以我们的 `:root` 为准，覆盖其默认 |
| Google Fonts 离线/被墙加载慢 | 本期用 CDN `@import`（匹配 `docs/04`）；阶段 6 打包时改自托管字体（记入 `docs/07` 后续） |
| 视差/呼吸动效卡顿 | 只动 `transform`/`opacity` + rAF 节流；reduced-motion 全停 |
| 端口冲突（5173 / 8787） | dev 端口可 `--port`；代理目标读 `AIDA_PORT` 约定（后端默认 8787） |

**文件清单**（`web/`）：

- 组件：`src/components/ui/button.tsx`（shadcn new-york + `accent` 变体）、`src/components/GlowOrb.tsx`（呼吸 keyframe + 视差，aria-hidden）。

**构建**：`npm run build`（`tsc -b && vite build`）零类型错误、零警告；产物 JS 270 KB（gzip 87 KB）、CSS 19 KB（gzip 4.4 KB）。

**真实验证**（Claude Preview，前端 5173 + 后端 `node src/http/server.ts` 同起）：
- 封面截图：雾白底 + 雾青呼吸光晕 + Lora 大标题「AI 对话档案」+ 青色「进入」CTA，符合 `docs/04 §4`。
- `/app` 经计算样式核对：body 背景 `rgb(245,245,240)`=`#F5F5F0`、body 字体 Raleway/Noto Sans SC、h1 字体 Lora；左右两栏 + 顶栏占位成立。
- **联通自检**：`/app` 头部显示「已连接 · Claude 47 · Codex 93」——前端经 Vite 代理打通后端 `/api/sources`，计数与阶段 2 实测一致。

**与计划的差异**：
- **`baseUrl` 去除**：TS 7.0 起 `baseUrl` 弃用（报 TS5101），改为仅 `paths`（相对 tsconfig 解析），别名仍可用。
- **字体 `@import` 顺序**：须置于 `@import 'tailwindcss'` 之前，否则 Tailwind 展开后 CSS 报「@import 必须在所有规则前」警告；已调整。
- **shadcn 用手写而非 CLI**：`shadcn init/add` 需联网且交互，按 docs/09 经验改为直接写 `components.json` + `lib/utils.ts` + `button.tsx`（标准 new-york 输出），加 `accent` 变体给 CTA。
- **crossfade 自检**：preview 的合成 click 未驱动 React 导航（用 `location.href` 直跳验证 `/app` 成立）；crossfade 逻辑（leaving→opacity→280ms navigate）已在 `Cover.tsx` 落地，reduced-motion 直接跳。
- **截图 networkidle 超时**：沙箱内 Google Fonts 请求挂起致 `/app` 截图等不到 idle 而超时；改用 `preview_inspect/eval` 核对 DOM 与计算样式，结论一致。生产/正常网络无此问题（阶段 6 可改自托管字体）。

**可达性自检**：焦点环青色 2px（`:focus-visible`）；reduced-motion 全局降级（呼吸/视差/crossfade 停）；CTA size=lg（h-44px）点击区充足；颜色对比墨字/雾白达标。
