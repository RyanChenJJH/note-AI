### 一键启动（推荐）
项目根 `start.cmd`：缺 `web/dist` 才构建 → 起服 → 自动开浏览器 → `http://127.0.0.1:8787`。
关服务：关掉最小化的 "AIDA · API" 窗口。

### 跑后端（`server/` 下，开发用）
```powershell
npm run serve                              # 起本地 API → http://127.0.0.1:8787（AIDA_PORT 可改）
node src/cli.ts scan                       # 列全部会话（CLI 自测）
node src/cli.ts show <id> [--thinking]     # 看分轮
node src/cli.ts export <id> --blocks r1-q,r1-a [--filename x]
node --test                                # 跑测试（63 个：内核 + HTTP + ai-skill/provider/parse/tidy + 静态/SPA）
```

### 跑前端（`web/` 下，开发用）
```powershell
npm install                                # 首次装依赖
npm run dev                                # Vite dev → http://localhost:5173（代理 /api→8787，需后端同起）
npm run build                              # tsc -b && vite build（类型+构建校验，产出 web/dist）
node --test                                # 跑前端纯逻辑测试（6 个：group + selection）
```
